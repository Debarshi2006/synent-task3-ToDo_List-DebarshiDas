📋 Task 3: To-Do List (CLI)

🚀 Purpose
This project is designed to implement an organized task manager that allows users to create, monitor, and mutate a personal list of objectives without interrupting the terminal runtime environment.

🛠️ Description
Developed as a continuous, menu-driven command-line program, this application focuses heavily on application control flow and in-memory data structures. It provides users with a structural control menu to append new items to an active task stack, view an indexed breakdown of current commitments, and delete completed tasks safely. The program contains built-in index validation logic to prevent data crashes and ensure seamless data manipulation within the command shell.
